# Route Optimization System

A full-stack Route Optimization System built using Spring Boot, React, PostgreSQL, and Graph Algorithms.

## Tech Stack

- Java 21
- Spring Boot
- React
- PostgreSQL
- Maven
- Leaflet

## Features

- Graph-based route optimization
- Dijkstra and A* pathfinding algorithms
- Traffic-aware routing
- Live traffic simulation using Spring Scheduler
- Real-time traffic updates with WebSockets
- Spring Cache for graph caching
- Interactive map visualization with Leaflet
- Algorithm performance statistics
- Responsive React dashboard
